# Talteen
Back up and transfer app data, documents, downloaded files, music, pictures and videos on your Sailfish OS devices. Available on [OpenRepos](https://openrepos.net/content/fravaccaro/talteen-backup).

* Back up Sailfish OS app data and files inside Documents, Downloads, Music, Pictures and Videos folders.
* Back up to an SD card.
* Restore from a backup.
* Over-the-air Wi-Fi transfer of backups.
