# Talteen
Backup and transfer app data, documents, music, pictures and videos on your Sailfish OS devices. Available on [OpenRepos](https://openrepos.net/content/fravaccaro/tarkka-magnifier).

* Backup of Sailfish OS app data and files inside Documents, Music, Pictures and Videos folders.
* Backup to SD card.
* Restore from a backup file.
* Over-the-air Wi-Fi transfer of data and files.
